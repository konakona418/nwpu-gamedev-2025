## Architecture

基于 Vulkan 1.3 开发的图形库，支持 2D、3D 渲染。使用 Slang 作为着色器编程语言。

基于 JoltPhysics 的物理系统。

基于 ENet 和 FlatBuffers 的网络通信。

基于 OpenAL 的音频。
